# 🌐 Sidhartha Kumar Malla — Data Analyst Portfolio

A modern, dark-themed portfolio website showcasing my projects, skills, certifications, and experience as a **Data Analyst**. Built with pure HTML, CSS, and JavaScript — featuring a live space animation background, smooth scroll reveals, and a responsive design.

## ✨ Features

- 🌌 **Live Space Animation** — Twinkling stars, shooting stars, nebula glows, and constellation connections
- 🎯 **Responsive Design** — Optimized for desktop, tablet, and mobile devices
- ⚡ **Smooth Animations** — Scroll-triggered reveal effects and micro-interactions
- 📊 **8 Featured Projects** — Data analytics and visualization projects with GitHub links
- 🛠️ **Comprehensive Skills** — 6 categorized skill sections covering the full data analytics stack
- 📜 **8 Certifications** — Professional certifications from IBM, HackerRank, and McKinsey
- 📬 **Contact Form** — Functional contact form powered by FormSubmit
- 🔝 **Back to Top** — Smooth scroll-to-top button

## 🚀 Getting Started

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/sidharthakumarmalla-lab/Portfolio.git
   ```

2. Open `index.html` in your browser, or serve locally:
   ```bash
   npx http-server -p 8080
   ```

3. Visit `http://localhost:8080`
